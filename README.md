## ▣ &nbsp;Featured Builds

<details open>
<summary><b>🧭 PathFinder — AI Admissions & Career Counseling Platform</b></summary>
<br/>

An AI-powered platform helping Pakistani students navigate university admissions and career paths.

- **Stack:** FastAPI · PostgreSQL · AI recommendation engine
- Production-grade documentation with a full developer manual
- Built to handle real student decision-making, not just a demo

</details>

<details>
<summary><b>📝 Bloggio — Full-Stack Real-Time Blogging Platform</b></summary>
<br/>

A blogging application built from the ground up, migrated from server-rendered EJS to a modern decoupled architecture.

- **Backend:** Node.js · Express · PostgreSQL · Socket.IO for real-time updates
- **Frontend:** React + Vite, talking to a thin JSON API layer
- Auth via httpOnly JWT cookies, clean `routes → controllers → models` architecture

</details>

<details>
<summary><b>🧩 xv6 Page Tables — OS Internals</b></summary>
<br/>

A RISC-V xv6 operating system assignment implementing and extending page table mechanics.

- Hands-on work with virtual memory, paging structures, and kernel-level OS concepts
- Part of ongoing systems-level coursework at ITU

</details>

<details>
<summary><b>💊 Medicare — Skincare Platform UI</b></summary>
<br/>

A skincare e-commerce-style website built as a single cohesive frontend experience.

- Clean, modern UI inspired by leading skincare platforms
- Focus on layout precision and responsive design
